2025/02/07よりTMPウェブサイトのマニュアルが断絶した上にアカウントが個人用で行われていた為以後引継ぎ用にHTML+CSSを用いて作成する。
しかし中京大学においては工学部、その中にメディア工学科こそあれど3DモデルやAutoCADといった自動車の設計側が多く、HTMLを周辺の大学でもあまり扱わないという事もありJavaは現段階では使用しないといういう方針になった。
放送クラブTMP並びに複数の部活で使用出来る人物を発掘・育成・継承によってHTMLテンプレート及び技術を継承し、今後の部活でも開発を続け、その人材を活用できるまでのルートを作成したい。
以上が計画概要で、今後の開発予定に関してを今から掲示する。

1-TMPの外注済みページとリンクをCABと連結、そして試験的動作の完了次第次の段階に移行する。
2-TMP定期企画の整理と独自リンクの配置、全プラットフォームで購読者の75%がどれか一つでしか見ておらず、既存のリピーターを他プラットフォームに移行させる媒体となる事で成功率が高く、また閲覧数を増加させる効果は大きいと見込みこれを最優先とする。
3-伊阪証のページ開発、現時点で視聴数比較ではYoutubeが一番多いが、伊阪証の小説ページの方がリピーターが多い。その為そこと連結し、2に組み込む。また、同人誌制作サークルとの協力でデザイン面を強化、逆に技術供与により相互関係の醸成を計画する。
4-現代社会学部は現在メディア業につく人物は多く、しかし同時にテレビメディアのトラブルにより今後減る可能性が高く、何より自前で発達した方が容易である。また、これを機にメディアを自社内に持つ・制作会社への直接委託が増えた影響で、この活動は予定外で遅くはなったが功を奏する可能性は高い。
5-YourDearMediaアーカイブ化、少し前まで印刷のエラーで中止したYourDearMediaをデジタルアーカイブとする。これはPhotoshopで行っていたがGithubでは重さに耐えれない為、手入力かAdobeの機能とDreamweaverを活用して開発を行う予定である。

他にも計画はあるが、試験的に行うべき事が多い上、マニュアルとして確立していない為暫くの間はHTMLのデバッグ、軽量化を進める。
